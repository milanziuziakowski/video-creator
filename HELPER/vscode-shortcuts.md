
# TOP (używam codziennie)
- Ctrl+P — Quick Open (otwórz plik po nazwie)
- Ctrl+Shift+P — Command Palette (wszystkie komendy)
- Ctrl+K, Ctrl+T — Zmiana motywu (Color Theme picker)
- Ctrl+` — Terminal wbudowany (pokaż/ukryj)
- Ctrl+B — Panel boczny (Explorer) pokaż/ukryj
- Ctrl+Shift+E — Explorer
- Ctrl+Shift+F — Global Search
- Ctrl+F — Szukaj w pliku
- Ctrl+H — Zamiana w pliku
- Ctrl+Shift+H — Zamiana w plikach
- Alt+Z — Zawijanie wierszy (Word Wrap)
- Ctrl+/ — Komentarz liniowy toggle
- Alt+↑ / Alt+↓ — Przesuń linię w górę/dół
- Shift+Alt+↑ / Shift+Alt+↓ — Duplikuj linię w górę/dół

## Motyw jasny/ciemny
- Ctrl+K, Ctrl+T — wybór theme
- (opcjonalnie) Settings → Window: Auto Detect Color Scheme — dopasuj do Windows

## Nawigacja po kodzie
- F12 — Go to Definition
- Alt+F12 — Peek Definition
- Shift+F12 — Find All References
- Ctrl+Shift+O — Outline symboli w pliku
- Ctrl+T — Go to Symbol in Workspace
- Ctrl+G — Go to Line

## Edycja
- Ctrl+D — zaznacz następne wystąpienie (multi-cursor)
- Ctrl+Shift+L — zaznacz wszystkie wystąpienia
- Alt+Click — multi-cursor myszą
- Ctrl+X — wytnij linię (nawet bez zaznaczenia)
- Ctrl+Enter — nowa linia poniżej
- Ctrl+Shift+Enter — nowa linia powyżej

## Refactor / format
- Shift+Alt+F — Format Document
- Ctrl+. — Quick Fix / Code Actions
- F2 — Rename symbol

## Okna / UI
- Ctrl+Shift+N — nowe okno
- Ctrl+W — zamknij kartę
- Ctrl+K, Ctrl+W — zamknij wszystkie karty
- Ctrl+Tab — przełącz kartę

## Git
- Ctrl+Shift+G — Source Control
